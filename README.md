# Dollet
Personal budget management application.

## Tech Stack

**Main:** .NET MAUI

**Dependencies:** CommunityToolkit.Mvvm 

## Features
- collecting data on expences and imcome
- ability to add valuable account ex. banks, investments, other
- handle currencies
- modern and light UI
